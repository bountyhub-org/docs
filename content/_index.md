# BountyHub

BountyHub is a platform designed to automate tool invocations and eliminate the tedious
aspects of bug bounty hunting. With the platform's assistance, you can focus on the
essential tasks at hand.

It eliminates the need to build your own pipeline and search for previously executed commands,
allowing you to concentrate more on the actual work that needs to be done.

This website intends to document features used across the
[bountyhub.org](https://bountyhub.org) platform.

{{< hint type=note >}}
The docs and the platform are in `alpha` version. They are subject to the rapid
change and based on your valuable feedback, it can be significantly improved.

It is, however, an attempt to describe intention and the inner-workings of the
platform, so it is easier for you to get started.
{{< /hint >}}
